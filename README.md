# Full Stack Movie Repository

### How to run
1. Open 3 terminals.
2. Make sure all of them are in the MovieTheater directory

Terminal 1
```bash
$ bash backend.sh
```

Terminal 2
```bash
$ cd frontend
$ ng serve
```

Terminal 3 <i>(Run this only when you want to terminate the app)</i>
```bash
$ bash terminate.sh
```
